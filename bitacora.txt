Linux only: libsecret
In Moodle App 3.9.3 we'll include a new script to easily push diff URLs to Moodle Tracker (Jira): gulp push. One of the libraries used requires the libsecret library to be installed in Linux before running npm install. Depending on your distribution, you will need to run the following command:

Debian/Ubuntu: sudo apt-get install libsecret-1-dev

despues de modificar las imagenes, escribir en consola, dentro de la carpeta moodleapp:

Para reconstruir la app con las modificaciones se utiliza:

ionic build

Para compilar la aplicacion en un ejectutable se utiliza (es necesario tener el sdk de android instalado) :

cordova platform remove android
cordova platform add android   o cordova platform add android@^9.0.0  (para el api 29 de android)


ionic cordova run android
